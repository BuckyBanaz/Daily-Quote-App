The Workflow

Step 1: ChatGPT

Ideation & Prompting

The foundation was laid by generating precise design prompts. ChatGPT helped define the UI direction, color palettes, and UX flows.

1. Foundation Design Prompt

Design a modern, minimal Daily Quote mobile app UI. Platform: Android & iOS (mobile-first) Style: Clean, calm, motivational, modern Color palette: Soft gradient background (light pastel tones), dark text for readability Typography: Sans-serif, clean, readable, inspirational feel Icon style: Rounded, minimal line icons Spacing: Comfortable, airy layout App should include: - Home screen with daily quote - Favorite quotes screen - Quote detail & share interaction - Empty states - Subtle micro-interactions Follow modern mobile UI/UX best practices.

2. Home Screen Prompt

Design the Home Screen of a Daily Quote App. Elements: - App bar with title "Daily Quote" and a heart (favorites) icon on the right - Centered quote card with: - Large quotation text - Author name below in smaller font - Soft card shadow and rounded corners - Background with subtle gradient - Action buttons below quote: - Favorite (heart icon) - Share (share icon) - Refresh quote (shuffle icon) - Bottom navigation bar with: - Home - Favorites UX: - Quote should feel calm and focused - Buttons should be clearly tappable - One-hand mobile usage friendly

3. Favorites Screen Prompt

Design the Favorites Screen for a Daily Quote App. Layout: - App bar with title "Favorites" - Scrollable list of saved quotes - Each quote in a rounded card with: - Quote text (2-3 lines max) - Author name - Small heart icon (filled) - Clean spacing between cards Empty State: - Illustration or icon - Text: "No favorites yet" - Subtext: "Tap the heart icon to save your favorite quotes" Style: - Same color palette and typography as Home Screen - Minimal and distraction-free

4. Share Sheet Prompt

Design a Share Quote Bottom Sheet. Components: - Drag handle at top - Preview of quote inside a card - Share options: - System Share - Copy Text - Close button Style: - Rounded top corners - Soft elevation - Clean and modern look - Focus on quote readability

5. Loading & Empty States

Design loading and empty states for a Daily Quote App. Loading State: - Minimal quote skeleton loader - Soft shimmer animation - Calm waiting experience Error State: - Friendly message: "Couldn't load quote" - Retry button Empty Favorites State: - Minimal illustration - Friendly motivational message

6. Micro-Interactions (Advanced)

Design subtle micro-interactions & animations to make the app feel alive. - Heart pulse on fav - Smooth fade on quote load - Staggered list entrance
